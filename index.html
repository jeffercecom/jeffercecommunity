CREAR UNA CUENTA DE WHATSAPP API EN 10 PASOS

1

Documentos requeridos para verificación Meta

  • Cámara de comercio
  • RUT
  • Factura de servicios públicos
2

Sitio web idóneo

  • Debe tener información legal visible
  • Políticas de privacidad, envíos, devoluciones, términos y condiciones
  • Footer con información completa
3

SIN SIM CARD nueva para crear WhatsApp API

  • No activa como plan pospago
  • Registrar con cédula del propietario
  • Debe tener 1 USD de saldo mínimo
  • No agregar a WhatsApp Business ni WhatsApp Personal
4

Dispositivos PC y Móvil

  • Agregar métodos de pago
  • Instalar Google Authenticator
  • Usar únicamente dispositivos del administrador
5

Correo Gmail del Admin

  • No debe ser corporativo
  • Debe estar disponible
  • Se usa para iniciar sesión en Meta y Google Chrome
6

Perfil de Facebook del Admin

  • Debe ser abierto desde PC
  • Contraseña disponible
  • Se liga al Gmail del paso 5
7

Abrir cuenta Pancake para Admin

  • Usar dispositivo del paso 4
  • Abrir desde Chrome con cuenta del paso 5
  • Crear cuenta en https://pm.pancake.biz
8

Abrir cuentas en ecosistema Pancake

  • Usar dispositivo del paso 4
  • Abrir en Chrome con correo Gmail del paso 5 y Facebook del paso 6
  • Crear cuentas en Poscake, Webcake y recargar la billetera Meta
9

Portafolios disponibles en Meta

  • Elegir el más idóneo
  • Activar Configuración WhatsApp API
  • Vincular número del paso 3
10

Crear API en Botcake

  • Abrir en dispositivo del paso 4
  • Login en https://botcake.io
  • Vincular billetera Meta creada
  • Crear la nueva cuenta API
